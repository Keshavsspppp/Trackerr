Button loading state
When "Save" is clicked, the button shows a spinner and becomes disabled. Currently re-clicking submits duplicate requests to the DB.
Easy
Status change animation
When a status badge is updated, it should briefly flash green before settling. Right now the update is instant and imperceptible — users don't know it worked.
Easy
Sortable column headers
Click "Company" to sort A→Z, click again for Z→A. A subtle ↑↓ arrow on hover shows the column is sortable. Pure client-side JS, no DB queries.
Easy
Drag to change status
Drag an application card between status columns (Kanban-style). Uses @dnd-kit/core. Converts the list view into a board view — toggle between the two.
Complex
Optimistic updates
Status changes and edits should reflect in the UI instantly, before the API confirms. Roll back on error. Eliminates the "did it save?" moment of doubt.
Medium
In-app notification dot
If an application hasn't moved in 7+ days, show a red dot on it in the list view — same trigger logic as the email reminder, but visible in-app too.
